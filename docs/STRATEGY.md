# 전략 문서

## Phase 1: RSI 기반 추세 전략

### 전략 개요

- **심볼**: BTC/USDT Perpetual Futures
- **타임프레임**: 1h (진입 시그널) + 4h (추세 필터)
- **지표**: RSI(14), EMA(50), EMA(200)

### 진입 조건

#### 롱 (매수)
1. 4h EMA50 > EMA200 (상승 추세 확인)
2. 1h RSI < 35 (과매도 — 눌림목)
3. 1h RSI 직전 봉 < 30 → 현재 봉 > 30 (상승 전환 확인)

#### 숏 (매도)
1. 4h EMA50 < EMA200 (하락 추세 확인)
2. 1h RSI > 65 (과매수 — 되돌림)
3. 1h RSI 직전 봉 > 70 → 현재 봉 < 70 (하락 전환 확인)

### 레인지장 필터 (진입 금지)
- 4h EMA50과 EMA200 간격이 0.5% 미만 → 횡보 구간으로 판단 → 진입 X

### 리스크 관리

| 항목 | 값 |
|---|---|
| 레버리지 | 3x (최대 5x) |
| 포지션 크기 | 시드의 20% 이하 |
| 손절 | 진입가 기준 -2% |
| 익절 | 진입가 기준 +4% (R:R = 1:2) |
| 일일 손실 한도 | 시드의 -30% → Kill Switch |

### 주문 실행 방식

```
진입 (MARKET)
  → 체결 확인
  → 손절: STOP_MARKET (reduceOnly=true)
  → 익절: TAKE_PROFIT_MARKET (reduceOnly=true)
```

Binance Futures는 OCO 미지원 → STOP + TAKE_PROFIT 동시 발주로 대체.
어느 쪽이든 먼저 체결되면 나머지는 자동 무효 (reduceOnly 특성).
